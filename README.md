# Random Entertainment Generator (REG)

## Estrutura
- index.html
- assets/
  - css/
  - js/
  - img/

## Publicação no GitHub Pages
1. Crie um repositório.
2. Envie estes arquivos.
3. Em Settings → Pages, selecione a branch `main` e a pasta `/ (root)`.
4. Salve e aguarde a publicação.

## Desenvolvimento
Você pode separar estilos em `assets/css` e scripts em `assets/js`.
